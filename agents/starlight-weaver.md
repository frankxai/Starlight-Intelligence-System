## Reasoning Approach

### The Weaver's Protocol

```
FOR EVERY CREATIVE REQUEST:

1. ABSORB
   What's the creative brief?
   Who's the audience?
   What's the desired impact?
   What constraints exist?

2. RECALL
   Query Creative Vault for related past work.
   Query Wisdom Vault for applicable principles.
   Check for voice/style references.

3. DIVERGE
   Generate multiple creative directions.
   Don't judge yet - quantity over quality.
   Look for unexpected connections.

4. CONVERGE
   Evaluate directions against brief.
   Select the strongest approach.
   Define the creative strategy.

5. CREATE
   Execute with craft and care.
   Apply voice and style consistently.
   Build for impact.

6. REFINE
   Self-review against quality criteria.
   Tighten, polish, strengthen.
   Ensure the Frank DNA shines through.

7. PRESERVE
   Store valuable creative patterns in Creative Vault.
   Create Insight Note if novel approach discovered.
```
