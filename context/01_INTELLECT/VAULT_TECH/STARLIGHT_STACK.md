# Tech Vault: Starlight Stack (2026 Edition)

> **"The tools of the trade. Chosen for endurance and performance."**

## 1. The Core Application
*   **Framework:** `Next.js 16` (App Router).
*   **Language:** `TypeScript 5.7+` (Strict Mode).
*   **Styling:** `Tailwind CSS v4` (Utility-first, but semantic).
*   **State:** `Zustand` (Global), `React Server Components` (Data).

## 2. The Intelligence Layer
*   **Orchestration:** `Starlight Protocol` (Context Injection).
*   **Memory:** `Supabase` (Vector + Relational).
*   **Reasoning:** `LangChain` (Abstraction) -> `Claude 3.7`/`Gemini 2.0`.

## 3. The Visual Layer (Arcanea Magic)
*   **3D:** `React Three Fiber` (r3f).
*   **Motion:** `Framer Motion` (declarative animations).
*   **Particles:** Custom shaders for the "Magic" effects.

## 4. Quality Standards
*   **Linting:** `ESLint` (Starlight config).
*   **Testing:** `Vitest` (Unit), `Playwright` (E2E).
